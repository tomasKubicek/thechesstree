Jak přispět?

Vzhledem k tomu, že je tato databáze otevřená komunitě, může do ní příspět každý.

Pokud chcete přispět, postupujte podle následujícího postupu:

  1. Nejdříve otevřete GitHub repositář tété webové stránky.
  2. Vytvořte fork repositáře na svůj účet.
  3. Najděte si otevření nebo koncovku, kterou budete chtít přidat.
  4. Vytvořte buď v "/data/otevreni/" nebo v "/data/koncovky/" soubor s příponou ".yml"
  5. Zkopírujte si šablonu "/data/template.yml" do souboru a vyplňte ji.
  6. Pošlete Pull Request do originální repositáře. Váš soubor se v databázi zobrazí, jakmile spojím váš pull request.
